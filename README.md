# 好伴AI · 竞品分析 + 增长方案 + 陪伴链 Demo

一个自包含的滚动叙事网站，把一份面向决策的竞品分析、一套增长方案，以及一个可交互的"陪伴链" Demo 合在一条主线里。结论基于 **2384 条应用商店评论聚类** 与 **好伴 / 蚂蚁阿福 / 小荷三家控制变量实测**。

## 在线查看

👉 **https://260926512.github.io/haoban-ai-case-study/**

## 内容

- **竞品分析**：四象限选品、6 考点实测、三家"记忆翻车"实测截图对比、连接器/陪伴者/工具战略路线、RAG vs 内生记忆架构、SOM 漏斗、"AI 替谁说话"立场、机会矩阵。
- **增长方案**：增长漏斗诊断（定位"留存"病灶）、北极星与增长公式、三阶段路线、需求优先级矩阵、三死结解法、家庭关系链"一鱼两吃"。
- **陪伴链 Demo**：手机模拟界面，5 屏可点 + 时间快进，体验"自动归档 → 真记忆 → 主动关怀"。
- **证据附录**：实测截图、评论数据、分析底稿，均可点开查看。

## 本地运行

纯前端、无依赖、离线可用。直接双击 `index.html`，或起一个本地服务器（图片路径在 `http://` 下更稳）：

```bash
python3 -m http.server 8848
# 打开 http://localhost:8848
```

> 所有数据与判断均可在站内"证据附录"追溯；测试体检报告为合成样本，姓名与数据均虚构。
